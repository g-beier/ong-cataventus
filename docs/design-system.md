# Design System — Cataventus

Apenas modo claro. Sem suporte a dark mode.

## Fontes

Comfortaa (Google Fonts) — `--font-comfortaa` / `font-sans`

## Cores

### Paleta da marca

| Token Tailwind        | Valor     | Descrição            |
|-----------------------|-----------|----------------------|
| `brand-cyan`          | `#00D9F6` | Ciano primário       |
| `brand-orange`        | `#F5701C` | Laranja — ação, CTA  |
| `brand-yellow`        | `#FEBD11` | Amarelo              |
| `brand-lime`          | `#C0DA1F` | Verde lima           |
| `brand-blue`          | `#A6DDF4` | Azul pastel          |
| `brand-pink`          | `#F8AED5` | Rosa pastel          |
| `brand-beige`         | `#E7E4C8` | Bege (centro logo)   |

### Tons neutros (apenas modo claro)

| Token Tailwind | Valor     | Uso                          |
|----------------|-----------|------------------------------|
| `ink`          | `#373435` | Texto principal              |
| `muted`        | `#373435` | Alias (compatibilidade)      |
| `ash`          | `#7A7476` | Texto secundário, legendas   |
| `canvas`       | `#FDFDFD` | Fundo da página              |
| `sheet`        | `#F5F2EF` | Cards, seções alternadas     |
| `rule`         | `#E5E0DD` | Bordas e divisores           |
